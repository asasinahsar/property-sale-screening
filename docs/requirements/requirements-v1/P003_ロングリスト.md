# 要件定義書: P003 ロングリスト

## 目的

担当者が選定した深掘り対象企業（ロングリスト）を管理し、責任者がレビュー・承認する。承認後の成果物として CSV／Excel でエクスポートし、上司への提出・持ち帰りに使う。

> **スコア呼称**: 構造スコア（=定量）／イベントスコア（=定性）／総合スコア（統合した派生値・並び順キー）。
> **粒度の注記**: 本フェーズは概念レベルのIPO＋UI状態遷移。具体的なテーブル名・API仕様は次のDB/API設計フェーズで追記する。Inputのスコア・指標・出典サマリの生成元は [`scoring-and-pipeline.md`](./scoring-and-pipeline.md) を参照。
> **実行権限**: 一覧閲覧・メモ編集・削除・エクスポートは **担当者・責任者ともに可**。**承認／却下は責任者ロールのみ**。
> **異常系の共通方針**: 処理失敗時は「エラー表示＋再試行導線」を共通パターンとする。
> **バリデーション**: 選定理由メモの最大文字数は500文字。エクスポートは対象0件時はボタン無効化＋「対象がありません」表示。

## 機能

| # | 機能 | 説明 | Phase | 仕様書からの展開／変更点 |
|---|------|------|-------|--------------|
| 1 | ロングリスト一覧表示 | 選定済み企業（企業名・構造/イベント/総合スコア・選定理由メモ・ステータス） | Phase1 | スコア呼称を統一 |
| 2 | 選定理由メモ編集 | 各企業に選定理由・補足メモを記入 | Phase1 | 保存タイミングを明記 |
| 3 | ステータス管理 | 候補／承認／却下のステータス | Phase1 | 状態遷移を明記 |
| 4 | レビュー・承認 | 責任者がレビューし承認／却下（責任者ロールのみ） | Phase1 | 認可制御を明記 |
| 5 | 企業削除 | リストから企業を除外 | Phase1 | 確認ダイアログを追加 |
| 6 | エクスポート | CSV／Excelでダウンロード（PowerPointはPhase2/任意） | Phase1 | 出力項目を明記（下記） |
| 7 | 詳細へ遷移 | 企業を選択しP002へ | Phase1 | 変更なし |

> **個社の根拠レポート出力（PDF/PPT）** は P002「根拠レポートのワンクリック出力」を参照する（本画面では重複定義しない）。

### エクスポート出力項目（Phase1）

企業名・証券コード・業種・構造スコア・イベントスコア・総合スコア・PBR・含み益（億）・含み益倍率・選定理由メモ・ステータス・出典サマリ。

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| ロングリスト一覧表示 | ロングリスト登録データ（スコアはscoring-and-pipeline.md由来） | 登録企業を取得し整形 | ロングリストテーブル |
| 選定理由メモ編集 | メモ本文（最大500文字） | 文字数バリデーション後に該当企業のメモを更新・保存 | 更新済みメモ／上限超過エラー |
| ステータス管理 | ステータス変更操作 | 候補→承認／却下へ更新（承認操作はロール検査） | 更新後ステータス（バッジ） |
| レビュー・承認 | 承認／却下操作（責任者） | ロールが責任者か検査→ステータス更新 | 承認済み／却下済み表示 |
| 企業削除 | 削除対象企業 | 確認後にロングリストから除外 | 更新後の一覧 |
| エクスポート | 出力形式（CSV／Excel） | 上記出力項目を整形しファイル生成。対象0件時は実行不可 | 成功：ダウンロードファイル／失敗：エラー表示＋再試行／0件：ボタン無効＋「対象がありません」 |
| 詳細へ遷移 | 企業選択 | 企業IDを保持して遷移 | P002 |

## UI状態遷移

| 要素 | 対応機能 | 状態 |
|------|---------|------|
| ロングリストテーブル | ロングリスト一覧表示 | ローディング／表示／0件（空状態） |
| メモ入力欄 | 選定理由メモ編集 | 通常／編集中／保存中／保存済み／上限超過エラー（500文字） |
| ステータスバッジ | ステータス管理 | 候補／承認／却下（色＋ラベル） |
| 承認・却下ボタン | レビュー・承認 | 表示（責任者）／非表示（担当者）／処理中 |
| 削除ボタン | 企業削除 | 有効／確認ダイアログ表示／削除中 |
| エクスポートボタン | エクスポート | 有効／生成中（ローディング）／完了／失敗（エラー表示＋再試行）／無効（対象0件） |
| 詳細リンク | 詳細へ遷移 | 有効 |
