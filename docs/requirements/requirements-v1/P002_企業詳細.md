# 要件定義書: P002 企業詳細（根拠レポート）

## 目的

1社について、構造スコア（定量）×イベントスコア（定性）を統合した総合スコアの内訳と、その根拠を出典付き（document_type・source_page・引用文）で確認する。担当者が「なぜこの企業が上位なのか」を理解し、追加観点の検討・最終判断につなげる。トレース可能率100%を満たす中核画面。

> **スコア呼称**: 構造スコア（=定量）／イベントスコア（=定性）／総合スコア（統合した派生値・並び順キー）。
> **粒度の注記**: 本フェーズは概念レベルのIPO＋UI状態遷移。具体的なテーブル名・API仕様・スコア計算式の係数は次のDB/API設計フェーズで追記する。Inputの「構造スコア・イベントスコア・各指標・定性シグナル・確信度」の生成元は [`scoring-and-pipeline.md`](./scoring-and-pipeline.md) を参照し、本画面は結果を受け取る前提。
> **実行権限**: 閲覧・根拠レポート出力・ロングリスト追加は **担当者・責任者ともに可**（ロール制限なし）。
> **異常系の共通方針**: 処理失敗時は「エラー表示＋再試行導線」を共通パターンとする。

## 機能

| # | 機能 | 説明 | Phase | 仕様書からの展開／変更点 |
|---|------|------|-------|--------------|
| 1 | 企業サマリ表示 | 企業名・証券コード・業種・時価総額等 | Phase1 | 変更なし |
| 2 | 総合スコア・内訳表示 | 総合スコアと構造／イベントの寄与内訳（レーダー＋数値併設） | Phase1 | スコア呼称を統一 |
| 3 | 定量指標明細（構造スコア根拠） | 標準セット（下記）の指標・該当数値・出典を表示 | Phase1 | 標準セットを明示 |
| 4 | ROIC vs WACC 対比表示 | 対象資産のROICとWACCを対比し、ROIC<WACC時に売却合理性が高い旨の根拠を表示 | Phase1 | 新規 |
| 5 | 定性シグナル一覧（イベントスコア根拠） | 売却シグナルを引用文付きで一覧。売却を否定する反証シグナルも区別して表示し、総合判定では減点根拠として扱う | Phase1 | 反証シグナルの区別を追加 |
| 6 | 確信度・データ充足度表示 | 各根拠に確信度／データ充足度を表示 | Phase1 | 新規 |
| 7 | 出典トレース | 各根拠に document_type・source_page・引用文を紐づけ。source_pageからEDINET等の開示原文の該当ページへ外部リンク | Phase1 | 外部リンクを追加・トレース可能率100%を明記 |
| 8 | AI総合判定 | AIの総合判定コメント（参照根拠に紐づけ） | Phase1 | 根拠参照を明記 |
| 9 | 根拠レポートのワンクリック出力 | 当画面の内容（スコア内訳・指標・出典付きシグナル・AI総合判定）を提案資料テンプレート（PDF/PowerPoint、最小はPDF）に流し込み出力 | Phase1 | 新規 |
| 10 | ロングリストへ追加 | この企業をP003へ追加 | Phase1 | 変更なし |
| 11 | 戻る | P001へ戻る | Phase1 | 変更なし |

### 定量指標明細の標準セット（Phase1）

売上高／PBR／修正PBR（含み益調整後）／自己資本比率／賃貸等不動産の時価・簿価・含み益／含み益／時価総額（含み益倍率）／対象資産のROIC・WACC（取得可能な場合）。各指標に該当数値と出典を紐づける。

## Input / Process / Output

| 機能 | Input | Process | Output |
|------|-------|---------|--------|
| 企業サマリ表示 | 企業ID | 企業基本情報を取得 | 企業ヘッダ |
| 総合スコア・内訳表示 | 企業のスコアリング結果 | 総合スコアと構造／イベント寄与を整形 | 総合スコアカード＋内訳（レーダー＋数値） |
| 定量指標明細 | 企業の構造スコア指標・財務数値 | 標準セットの指標ごとにスコア寄与・該当数値・出典を対応付け | 定量指標テーブル |
| ROIC vs WACC 対比表示 | 対象資産のROIC・WACC | 両者を対比し、ROIC<WACC を判定 | 対比表示＋売却合理性の根拠 |
| 定性シグナル一覧 | 企業の定性シグナル（賛否） | 売却支持／反証シグナルを区別し引用文・出典を整理 | 定性シグナルカード群（支持/反証） |
| 確信度・データ充足度表示 | 企業の確信度／データ充足度 | 各根拠に紐づけ | 確信度表示 |
| 出典トレース | 根拠（指標・シグナル） | document_type・source_page・引用文を付与し外部リンクを生成。リンク切れ時はフォールバック | 出典詳細＋開示原文への外部リンク／リンク無効時は「リンク無効」表示（document_type・source_page・引用文は保持） |
| AI総合判定 | スコア・根拠一式 | AI判定コメントを生成・参照根拠に紐づけ | AI判定パネル |
| 根拠レポート出力 | 出力形式（PDF/PPT）、当画面の内容 | テンプレートに流し込み生成 | 提案資料ファイル（最小はPDF） |
| ロングリストへ追加 | 企業ID | ロングリストに登録（重複排除） | P003へ追加・完了通知 |
| 戻る | 戻る操作 | 直前一覧の状態を保持して遷移 | P001 |
