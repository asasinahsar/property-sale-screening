## 主要なID（識別子）の関係

- **ユーザーID(1)** … ロングリスト登録者/承認者(61/62)、ウォッチ対象者の主体
- **企業ID(10)** … 財務/市場(3章)・開示文書(28経由のシグナル)・スコア(41)・イベント(56)・ロングリスト(58)・ウォッチ(72) から参照される中心エンティティ
- **文書ID(28)** … 定性シグナル(40)・出典トレース(D7)が参照（トレーサビリティの起点）
- **スクリーニング実行ID(50)** … 各企業のスコアリング結果(41)を1実行単位で束ねる
- **証券コード(11)** は外部データ突合の自然キー、**企業ID(10)** は内部の主キー（重複ID混在を避けるため内部IDを正とする）
