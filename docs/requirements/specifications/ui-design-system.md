## スタイル方針

- **スタイル**: Data-Dense Dashboard（BI／金融分析／エンタープライズ・レポーティング向け）
- **特徴**: 複数ウィジェット・データテーブル・KPIカード・最小パディング・グリッドレイアウトで情報密度を最大化
- **アクセシビリティ**: WCAG AA / パフォーマンス良好
- **補足**: MASTER.md の Page Pattern「Enterprise Gateway」は対外ランディング向けのため、**本プロダクト（社内ツール）では採用しない**。スタイル・カラー・タイポグラフィ・コンポーネント仕様のみ適用する。

## コンポーネント（MASTER.md準拠）

- **ボタン**: Primary＝アンバー塗り（実行系）、Secondary＝ブルー枠線。`transition 200ms`、`cursor-pointer`、非同期処理中は disable
- **カード**: 角丸12px・`shadow-md`、hoverで`shadow-lg`＋わずかな浮き（レイアウトシフトしない）
- **入力**: focus時にPrimaryブルーのフォーカスリングをWCAG準拠で表示
- **モーダル**: オーバーレイ`rgba(0,0,0,0.5)`＋blur、確認ダイアログ等に使用
- **z-indexスケール**: 10 / 20 / 30 / 50 の体系を使用（`z-[9999]`のような任意大値は禁止）

## 主要UXルール（ux/webドメイン）

- **テーブルのはみ出し**: `overflow-x-auto` ラッパーで対応（モバイルで横スクロール、または重要列のカード化）
- **一括操作**: ランキング/ロングリストはチェックボックス列＋アクションバーで複数選択・一括追加/削除（1行ずつの操作にしない）→ P001「ロングリストに追加」、P003一括承認に適用
- **ローディング**: スクリーニング実行・再読込時はスケルトン/スピナーで状態提示。非同期完了まで実行ボタンを無効化
- **コンテンツジャンプ防止**: 非同期コンテンツ領域は事前に高さを確保
- **アクセシビリティ**: アイコンのみボタンは`aria-label`、フォームは`label`必須、フォーカス可視、`prefers-reduced-motion`順守、色のみに依存しない

## ページ別の適用メモ

- **P000 ログイン**: 中央寄せのシンプルフォーム。Primaryフォーカスリング、認証中はボタン無効化、エラーは入力近傍に表示
- **P001 ダッシュボード**: 上部KPIカード4枚＋直近イベントバナー → フィルタ → ランキングテーブル（行hoverハイライト、チェックボックス＋アクションバー、ソート）。実行/再読込はAccentのPrimaryボタン
- **P002 企業詳細**: スコア内訳はレーダー＋数値テーブル併設。定性シグナルはカード、出典（document_type・source_page・引用文）をhoverツールチップ/展開で表示
- **P003 ロングリスト**: ステータスはバッジ＋ラベル（色のみに依存しない）。承認/却下は責任者ロールのみ表示。CSV/Excelエクスポートボタンはツールバー
