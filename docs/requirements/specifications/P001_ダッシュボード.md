# 仕様書: P001 ダッシュボード（兼 企業ランキング一覧）

## 目的

ログイン後の入口画面。KPIカードと直近イベントで全体を把握しつつ、構造スコア（定量）×イベントスコア（定性）を統合した企業ランキングを閲覧・絞り込みし、深掘り対象（ロングリスト）の選定を開始する。スクリーニングの実行／再読込もここから行う。評価・ランキングの単位は「企業」。

> 詳細なIPO・UI状態遷移・実行権限は `requirements-v1/P001_ダッシュボード.md` を参照。スコア等の生成元は `requirements-v1/scoring-and-pipeline.md`。

## 機能一覧

| # | 機能 | 説明 | Phase |
|---|------|------|-------|
| 1 | KPIカード表示 | 対象企業数・高構造スコア社数（しきい値=構造スコア70以上）・平均構造スコア・直近イベント件数 | Phase1 |
| 2 | 直近イベントバナー | 直近7日の新規イベント企業／イベントスコア（Phase1はスナップショット） | Phase1 |
| 3 | スクリーニング実行／再読込 | 一括スコアリングを起動しランキング・KPIを更新（担当者・責任者とも実行可） | Phase1 |
| 4 | 企業ランキング表示 | 確定列で総合スコア順に企業を一覧表示 | Phase1 |
| 5 | フィルタ・並び替え | 業種・時価総額帯・スコア帯・イベント有無で絞り込み。並び替えトグル（総合／構造／イベント、既定=総合） | Phase1 |
| 6 | 企業検索（完全一致＋自然言語） | 企業名/証券コードの完全一致に加え、自然言語クエリをAIエージェントが解釈する自然言語検索モード | Phase1 |
| 7 | 詳細へ遷移 | 行選択でP002へ | Phase1 |
| 8 | ロングリストへ追加 | チェックした企業をP003へ一括追加 | Phase1 |
| 9 | 確信度インジケータ | データ充足度が低い企業をスコアセルに控えめなバッジ/トーンでインライン表示 | Phase1 |
| 10 | イベントブースト倍率バッジ | イベント加点倍率（例「×1.30」）を企業名横に表示 | Phase1 |
| 11 | スコア変化の差分ハイライト | 前回スクリーニング比でスコア上昇/下降を矢印・色で強調 | Phase2 |
| 12 | ウォッチ登録動線 | 企業をP004（通知・ウォッチリスト）へ登録 | Phase2 |

### ランキングテーブル表示列（確定・Phase1）

企業名（行ラベル/左端・倍率/低確信度バッジをインライン）／業種／構造スコア／イベントスコア／PBR／含み益（億）／イベント／詳細ボタン。

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| KPIカード群 | 4枚のサマリカード | 画面上部 |
| 新規イベントバナー | 直近7日の新規イベント企業／イベントスコア | KPIカード直下 |
| 実行／再読込ボタン | 「スクリーニング実行」「再読込」「イベント更新」 | 上部右ツールバー |
| フィルタパネル | 業種・時価総額・スコア帯・イベント有無 | ランキング上部／左サイド |
| 並び替えトグル | 総合／構造／イベント の切替 | テーブルヘッダ付近 |
| 検索ボックス | 完全一致／自然言語モード切替 | フィルタ付近 |
| ランキングテーブル | 確定列＋選択チェックボックス。スコアセルに確信度インジケータ、企業名横にブースト倍率バッジ | 画面中央〜下部 |
| 差分インジケータ | スコア上昇/下降の矢印・色（Phase2） | スコアセル内 |
| 「ロングリストに追加」ボタン | 選択企業をP003へ | テーブル上部/下部 |
| ページネーション | 一覧の分割表示 | テーブル下部 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| KPIカード表示 | KPIカード群 | 表示時／再読込時に最新値で更新 |
| 直近イベントバナー | 新規イベントバナー | 「イベント更新」押下で更新（Phase1はスナップショット） |
| スクリーニング実行／再読込 | 実行／再読込ボタン | 押下で一括スコアリング起動→テーブル・KPI更新（失敗時はエラー＋リトライ） |
| 企業ランキング表示／並び替え | ランキングテーブル／並び替えトグル | 既定は総合スコア降順、トグルで構造/イベント順 |
| フィルタ | フィルタパネル | 条件変更で再描画 |
| 企業検索（完全一致＋自然言語） | 検索ボックス | 完全一致／自然言語クエリを解釈し絞り込み＋要約（解釈失敗時は専用表示） |
| 確信度インジケータ | スコアセル | 低確信度をインライン表示 |
| イベントブースト倍率バッジ | 企業名横バッジ | 倍率を表示 |
| スコア差分ハイライト（Phase2） | 差分インジケータ | 前回比で上昇/下降を強調 |
| 詳細へ遷移 | テーブル行／詳細ボタン | クリックで P002 |
| ロングリストへ追加 | チェックボックス＋追加ボタン | チェック後押下で P003 へ追加 |
| ウォッチ登録動線（Phase2） | 行アクション | P004 へ登録 |
