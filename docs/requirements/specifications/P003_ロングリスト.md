# 仕様書: P003 ロングリスト

## 目的

担当者が選定した深掘り対象企業（ロングリスト）を管理し、責任者がレビュー・承認する。承認後の成果物として CSV／Excel でエクスポートし、上司への提出・持ち帰りに使う。

> 詳細なIPO・UI状態遷移・実行権限は `requirements-v1/P003_ロングリスト.md` を参照。スコア等の生成元は `requirements-v1/scoring-and-pipeline.md`。
> 個社の根拠レポート出力（PDF/PPT）は P002「根拠レポートのワンクリック出力」を参照（本画面では重複定義しない）。

## 機能一覧

| # | 機能 | 説明 | Phase |
|---|------|------|-------|
| 1 | ロングリスト一覧表示 | 企業名・構造/イベント/総合スコア・選定理由メモ・ステータス | Phase1 |
| 2 | 選定理由メモ編集 | 各企業に選定理由・補足メモを記入（最大500文字） | Phase1 |
| 3 | ステータス管理 | 候補／承認／却下のステータス | Phase1 |
| 4 | レビュー・承認 | 責任者がレビューし承認／却下（**責任者ロールのみ**） | Phase1 |
| 5 | 企業削除 | リストから企業を除外（確認ダイアログ） | Phase1 |
| 6 | エクスポート | CSV／Excelでダウンロード（担当者・責任者とも実行可。PowerPointはPhase2/任意） | Phase1 |
| 7 | 詳細へ遷移 | 企業を選択しP002へ | Phase1 |

### エクスポート出力項目（確定・Phase1）

企業名・証券コード・業種・**構造スコア・イベントスコア・総合スコア**・PBR・**含み益（億）・含み益倍率**・選定理由メモ・ステータス・**出典サマリ**。

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| ロングリストテーブル | 企業名・各スコア・選定理由メモ・ステータス・操作 | 画面中央 |
| ステータスバッジ | 候補/承認/却下を色＋ラベルで表示 | 各行 |
| メモ入力欄 | 選定理由・補足（最大500文字） | 各行（インライン/モーダル） |
| 承認・却下ボタン | 責任者のレビュー操作（権限により表示） | 各行/一括操作バー |
| 削除ボタン | 企業をリストから除外 | 各行 |
| エクスポートボタン | CSV／Excel出力 | テーブル上部ツールバー |
| 詳細リンク | 企業名からP002へ | 各行 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| ロングリスト一覧表示 | ロングリストテーブル | 表示時に選定済み企業をロード（0件は空状態） |
| 選定理由メモ編集 | メモ入力欄 | 入力・保存（500文字上限バリデーション） |
| ステータス管理 | ステータスバッジ | 承認/却下に連動して更新 |
| レビュー・承認 | 承認・却下ボタン | 責任者ロールのみ押下可 |
| 企業削除 | 削除ボタン | 確認後にリストから除外 |
| エクスポート | エクスポートボタン | 押下でCSV/Excel生成（生成中/完了/失敗、対象0件は無効化） |
| 詳細へ遷移 | 詳細リンク | クリックでP002へ |
