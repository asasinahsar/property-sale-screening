# 仕様書: P002 企業詳細（根拠レポート）

## 目的

1社について、構造スコア（定量）×イベントスコア（定性）を統合した総合スコアの内訳と、その根拠を出典付き（document_type・source_page・引用文）で確認する。担当者が「なぜこの企業が上位なのか」を理解し、追加観点の検討・最終判断につなげる。トレース可能率100%を満たす中核画面。

> 詳細なIPO・UI状態遷移・実行権限は `requirements-v1/P002_企業詳細.md` を参照。スコア等の生成元は `requirements-v1/scoring-and-pipeline.md`。

## 機能一覧

| # | 機能 | 説明 | Phase |
|---|------|------|-------|
| 1 | 企業サマリ表示 | 企業名・証券コード・業種・時価総額など | Phase1 |
| 2 | 総合スコア・内訳表示 | 総合スコアと構造／イベントの寄与内訳（レーダー＋数値併設） | Phase1 |
| 3 | 定量指標明細（構造スコア根拠） | 標準セット（売上高/PBR/修正PBR/自己資本比率/賃貸等不動産 時価・簿価・含み益/含み益倍率/ROIC・WACC）と該当数値・出典 | Phase1 |
| 4 | ROIC vs WACC 対比表示 | ROICとWACCを対比し、ROIC<WACC時に売却合理性が高い旨の根拠を表示 | Phase1 |
| 5 | 定性シグナル一覧（イベントスコア根拠） | 売却支持シグナルと反証シグナル（売却否定）を区別して引用文付きで表示 | Phase1 |
| 6 | 確信度・データ充足度表示 | 各根拠に確信度／データ充足度を表示 | Phase1 |
| 7 | 出典トレース | document_type・source_page・引用文を紐づけ、EDINET等の開示原文へ外部リンク | Phase1 |
| 8 | AI総合判定 | AI総合判定コメント（参照根拠に紐づけ） | Phase1 |
| 9 | 根拠レポートのワンクリック出力 | 当画面の内容を提案資料テンプレート（PDF/PowerPoint、最小PDF）に出力（担当者・責任者とも実行可） | Phase1 |
| 10 | ロングリストへ追加 | この企業をP003へ追加 | Phase1 |
| 11 | 戻る | P001へ戻る | Phase1 |
| 12 | ウォッチ登録動線 | この企業をP004へ登録 | Phase2 |

## UI要素

| 要素 | 説明 | 位置 |
|------|------|------|
| 企業ヘッダ | 企業名・証券コード・業種・時価総額 | 画面上部 |
| 総合スコアカード | 総合スコアと評価ランク | ヘッダ直下 |
| スコア内訳チャート | 構造／イベントの寄与（レーダー）＋数値 | 総合スコアカード横/下 |
| 定量指標テーブル | 指標名・数値・スコア寄与・出典 | 中央左 |
| ROIC/WACC対比 | ROICとWACCの対比表示 | 定量指標付近 |
| 定性シグナルカード | シグナル種別（支持/反証）・引用文・出典 | 中央右〜下部 |
| 確信度表示 | 根拠ごとの確信度／データ充足度 | 各根拠付近 |
| 出典欄／外部リンク | document_type・source_page・引用文＋開示原文リンク | 各根拠 |
| AI判定パネル | AI総合判定コメント | 上部/右サイド |
| レポート出力ボタン | PDF/PPT出力 | ヘッダ右/下部 |
| ロングリスト追加ボタン | P003へ追加 | ヘッダ右/下部 |
| 戻るボタン | P001へ戻る | ヘッダ左 |

## 機能とUIの結びつき

| 機能 | UI要素 | インタラクション |
|------|--------|----------------|
| 企業サマリ表示 | 企業ヘッダ | 表示時ロード（失敗時エラー） |
| 総合スコア・内訳表示 | 総合スコアカード／内訳チャート | 表示時に描画 |
| 定量指標明細 | 定量指標テーブル | 各行に数値・出典 |
| ROIC vs WACC 対比 | ROIC/WACC対比 | ROIC<WACC時に強調（取得不可時は非表示） |
| 定性シグナル一覧 | 定性シグナルカード | 支持/反証を区別表示 |
| 確信度・データ充足度 | 確信度表示 | 高/中/低を表示 |
| 出典トレース | 出典欄／外部リンク | 展開で出典表示、リンクで開示原文へ（切れ時はフォールバック） |
| AI総合判定 | AI判定パネル | 表示時ロード |
| 根拠レポート出力 | レポート出力ボタン | 押下でPDF/PPT生成（生成中/完了/失敗） |
| ロングリストへ追加 | ロングリスト追加ボタン | 押下でP003へ追加 |
| 戻る | 戻るボタン | 押下でP001へ |
| ウォッチ登録動線（Phase2） | 行/ヘッダアクション | P004へ登録 |
